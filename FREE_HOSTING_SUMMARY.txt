                     🚀 DEPLOY TO RENDER IN 3 STEPS
================================================================================

STEP 1: Push to GitHub (2 minutes)
---------------------------------------
Option A - GitHub Desktop (easiest):
1. Download: https://desktop.github.com/
2. Open GitHub Desktop
3. Add repository: D:\nutrition_diet_expert_system-main
4. Click "Publish repository"
5. Done!

Option B - Command Line:
> cd D:\nutrition_diet_expert_system-main
> git init
> git add .
> git commit -m "Ready for deployment"
> git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
> git push -u origin main

STEP 2: Deploy to Render (2 minutes)
---------------------------------------
1. Go to: https://render.com
2. Sign up (free, use GitHub)
3. Click "New +" → "Web Service"
4. Connect your GitHub repository
5. Render reads render.yaml and configures automatically
6. Click "Create Web Service"
7. Wait 5-10 minutes

STEP 3: Add Database & Seeds (1 minute)
---------------------------------------
1. Render creates PostgreSQL automatically (from render.yaml)
2. After deployment, open "Shell" in Render dashboard
3. Run: python seeds/seed.py restore
4. Done!

Your app is LIVE at: https://nutrition-diet-expert.onrender.com

================================================================================
                          🆘 TROUBLESHOOTING
================================================================================

Problem: Build failed
Solution: Check Render logs, ensure all requirements installed

Problem: Database connection error
Solution: Verify DATABASE_URL is set (Render does this automatically)

Problem: App not loading
Solution: Check logs in Render dashboard for Python errors

Problem: Slow first load
Solution: Normal on free tier - app sleeps after inactivity

Problem: Need to reset database
Solution: In Render shell: python seeds/seed.py restore
